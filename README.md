# ColorPicker
Design a simple user interface (UI) and implement specific functionality based on the given requirements.

## Tech Stack
- TypeScript
- Vuetify
- Vite

## How to use
1. Clone the repo
``` bash
git clone https://github.com/marahtadeo/ColorPicker.git
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm run dev
```
